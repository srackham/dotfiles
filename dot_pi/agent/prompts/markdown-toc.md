description: Update and validate the table of contents and links in a Markdown file.
argument-hint: "<file name>"
---
In the $1 Markdown file I want you to:

- Update the table of contents, or add a table of contents if there is none.
- Only include level 2 sections (i.e. titles beginning with two hashes) in the table of contents.
- Validate and if necessary update internal links.
- Create new links when you find references (implicit or explicit) to sections elsewhere in the document.
- Remove unresolvable links.
